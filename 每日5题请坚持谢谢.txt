3.说说&&和&区别
&&和&都作为逻辑运算符,当&&或者&只要一边出现了false都不会执行满足条件的代码，只有当两边都出现了true才会执行满足条件的代码
不同点:
&&还具有短路功能,当&&左边出现了false时,它不再执行右边的表达式,只有当它两边都出现了true才会执行满足条件的表达式
&左边出现了false时，它还是会执行右边的表达式,它还可以当作位运算符,任何字节和oxof相与，都可以获取到它的第四位
逻辑操作(&,|,^)和条件操作(&&,||)区别
条件操作(逻辑运算符)只能操作布尔类型,而逻辑操作还可以操作数值类型
逻辑操作不产生短路

5.swich能否作用于byte，long，string上?
switch是可以作用于byte上的,不能作用于long上.在switch（cc）中必须满足cc是int类型或者是integer包装类型或者是枚举类型.byte，short,char类型都可以隐士转换成int类型，所以他们是可以的,但是long必须强
转成int类型,所以long不行
在jdk7以后中switch是支持string类型

7.char型变量中能不能存储一个中文汉字为什么?
char类型变量储存的是一个unicod字符集,而unicode字符集是能储存中文汉字的，所以char类型变量也是能储存中文汉字的,对于一些生僻的中文汉字，unicode中是没有的，所以char中自然也是没有的储存的，对于一个中文汉字在unicode中占两个字节
所以char中也是占两个字节

11.==和equals有什么区别
==是比较两个数据在内存中的值是否相等
equal()是比较两个相互独立的对象的内容
String a=new String("foot");
String b=new String("foot");
a==b;false
a.equals(b);true
分别创建了两个对象，在栈内存中储存了ab两个地址，他们都指向堆中的同一个“foot”值所以a==bfalse a.equal(b)true
12静态变量和实例变量的区别？
在语法上：今天变量是通过static,实例变量一般没有
在运行上:静态变量在类加载的时候就创建了，而实例必须要创建对象才能创建，静态变量可以通过类名直接类名.变量名.而实例变量
必须创建对象，通过对象才能引用它的实例变量

24abstract class 和interface有什么区别?
不同:
abstract class中有普通方法也有抽象方法，必须在子类中被实现抽象方法，或者子类也是一个抽象类
接口 中方法都是抽象的，必须在实现类中实现所有的抽象方法
相同:
都不能创建实例，都含有抽象方法

应用区别:
接口主要实现系统设计模块之间的通信锲约。而抽象类主要作用是将一些公共的成员抽取实现代码重用
